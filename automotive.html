🤖 AI Identification — Anthropic API Key

The AI photo identification uses Claude (by Anthropic). Enter your API key below — it is stored securely in your Firebase account and shared across the Farmboss Headquarters.

Get your API key at console.anthropic.com → API Keys → Create Key

🌐 CORS Proxy (Required for AI on GitHub Pages)

GitHub Pages blocks direct calls to the Anthropic API (CORS policy). A free Cloudflare Worker proxy fixes this. Set up the proxy in Farmboss Armory Settings first — the same Worker URL works for all apps.

After saving, click Test Connection above to verify.

📷 Firebase Storage Setup

Photo uploads use Firebase Storage — shared across the Farmboss Headquarters. If already set up in Armory, photos will work here automatically.

  1. Go to console.firebase.google.com → your watchroom project
  2. Click Storage in the left sidebar → Get started
  3. Choose Start in production mode → Next → pick your region → Done
  4. Click the Rules tab and replace all content with:
    rules_version = '2'; service firebase.storage { match /b/{bucket}/o { match /{allPaths=**} { allow read, write: if request.auth != null; } } }
