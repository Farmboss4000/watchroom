📷 Firebase Storage Setup

Photo uploads use Firebase Storage. Enable it once and photos will work across all Farmboss apps.

  1. Go to console.firebase.google.com → your watchroom project
  2. Click Storage in the left sidebar → Get started
  3. Choose Start in production mode → Next → pick your region → Done
  4. Click the Rules tab and replace all content with:
    rules_version = '2'; service firebase.storage { match /b/{bucket}/o { match /{allPaths=**} { allow read, write: if request.auth != null; } } }
🔒 Firestore Rules

Ensure your Firestore rules allow authenticated access:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }
