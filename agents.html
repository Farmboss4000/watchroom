📖 How Streaming Works

The Run Console connects to the Anthropic API via your Cloudflare Worker proxy and streams the response token by token using Server-Sent Events (SSE).

The proxy forwards stream: true to the Anthropic API. As each token arrives, it appears immediately in the console — just like in Claude.ai.

If streaming fails, the console automatically falls back to a standard (non-streaming) request and displays the full response when complete.
